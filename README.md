## 7 Core Features:

### 1. **JSON Formatter**
- Format/prettify JSON with proper indentation
- Minify JSON to single line
- Real-time syntax validation
- Error messages for invalid JSON

### 2. **Base64 Encoder/Decoder**
- Encode text to Base64 format
- Decode Base64 strings back to text
- Error handling for invalid Base64

### 3. **Hash Generator**
- Generate MD5 hashes
- Generate SHA-1 hashes
- Generate SHA-256 hashes
- All hashes from single input

### 4. **Color Converter**
- Convert HEX to RGB format
- Convert HEX to HSL format
- Live color preview box
- Supports all HEX color codes

### 5. **URL Encoder/Decoder**
- Encode URLs and special characters
- Decode URL-encoded strings
- Perfect for query parameters
- Error handling for malformed URLs

### 6. **JWT Decoder**
- Decode JWT tokens
- View header information
- View payload data
- See signature hash
- Note: Decodes only, doesn't verify signatures

### 7. **Timestamp Converter**
- Convert Unix timestamp to date/time
- Convert date/time to Unix timestamp
- Get current timestamp instantly
- Shows multiple formats (UTC, Local, ISO)
- Supports both seconds and milliseconds
